# React Query - Movie Search Application

Приложение для поиска фильмов с использованием TMDB API, интеграцией React Query для управления состоянием запросов и пагинацией результатов.

## Описание

Приложение позволяет пользователям:
- Искать фильмы по ключевому слову
- Просматривать результаты в виде сетки карточек фильмов
- Переходить между страницами результатов
- Просматривать детальную информацию о фильме в модальном окне
- Получать уведомления о состоянии приложения

## Технологический стек

- React 19+
- TypeScript
- Vite
- CSS Modules
- TanStack Query (React Query)
- React Paginate
- Axios
- React Hot Toast

## Установка и запуск

### Требования
- Node.js 16+
- npm или yarn

### Шаги установки

1. Клонирование репозитория
```bash
git clone <repository-url>
cd 04-react-query
```

2. Установка зависимостей
```bash
npm install
```

3. Настройка переменной окружения

Создайте файл `.env.local` в корне проекта:
```
VITE_TMDB_TOKEN=ваш_токен_от_TMDB
```

Как получить токен от TMDB:
1. Перейдите на https://www.themoviedb.org/
2. Создайте или войдите в аккаунт
3. Перейдите в Settings → API
4. Скопируйте Bearer Token

Для развертывания на Vercel добавьте переменную окружения через дашборд:
1. Settings → Environment Variables
2. Добавьте переменную `VITE_TMDB_TOKEN` с вашим токеном

4. Запуск dev-сервера
```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173/`

5. Сборка для production
```bash
npm run build
```

6. Предпросмотр production-сборки
```bash
npm run preview
```

## Основные компоненты

### SearchBar
- Форма поиска с валидацией
- Проверка на пустое значение
- Уведомление об ошибке при попытке поиска без текста

### MovieGrid
- Сетка карточек фильмов
- Ленивая загрузка изображений
- Обработка клика для открытия детальной информации

### Loader
- Индикатор загрузки данных
- Отображается во время запроса к API

### ErrorMessage
- Уведомление об ошибке
- Показывается при ошибке запроса к API

### MovieModal
- Модальное окно с информацией о фильме
- Закрывается при клике на кнопку, клавишу ESC или клике за пределами окна
- Использует React Portal для рендеринга
- Блокирует прокрутку страницы при открытии

## React Query и пагинация

Приложение использует TanStack Query для:
- Кэширования результатов поиска
- Управления состоянием загрузки и ошибок
- Повторных попыток при ошибках
- Синхронизации состояния между компонентами

Пагинация реализована через React Paginate:
- Отображает номера страниц
- Показывает кнопки "назад" (←) и "вперед" (→)
- Кэшируется отдельно для каждого набора параметров поиска
- Видна только при наличии более одной страницы результатов

## API интеграция
```
GET https://api.themoviedb.org/3/search/movie
Приложение использует The Movie Database (TMDB) API для получения информации о фильмах.

Конечная точка:
```
GET https://api.themoviedb.org/3/search/movie
```

Параметры:
- `query` - текст для поиска
- `include_adult` - включать контент для взрослых
- `language` - язык результатов
- `page` - номер страницы результатов

Авторизация: Bearer Token через заголовок Authorization
