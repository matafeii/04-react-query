# React Movies - Movie Search Application

Застосунок для пошуку фільмів за допомогою TMDB API з сучасним інтерфейсом на React та TypeScript.

🌐 **[Live Demo на Vercel](https://03-react-movies-teal-seven.vercel.app/)**

## 📋 Описання

**React Movies** - це повнофункціональна веб-програма для пошуку фільмів. Користувачі можуть:
- 🔍 Шукати фільми за ключовим словом
- 🎬 Переглядати карточки фільмів у галереї
- 📖 Відкривати детальну інформацію в модальному вікні
- ⚡ Отримувати миттєвий зворотний зв'язок через сповіщення

## 🛠️ Технологічний стек

- **React** 18+ - бібліотека для побудови UI
- **TypeScript** - типізація JavaScript
- **Vite** - сучасний build-інструмент
- **CSS Modules** - стилізація з ізоляцією область видимості
- **Axios** - HTTP-клієнт для API запитів
- **React Hot Toast** - система сповіщень
- **modern-normalize** - нормалізація стилів браузерів
- **Prettier** - форматування коду

## 🚀 Встановлення та запуск

### Передумови
- Node.js (версія 16+)
- npm або yarn

### Кроки встановлення

1. **Клонування репозиторію**
```bash
git clone <repository-url>
cd 03-react-movies
```

2. **Встановлення залежностей**
```bash
npm install
```

3. **Налаштування змінної оточення**

Створіть файл `.env.local` в корені проєкту:
```
VITE_TMDB_TOKEN=вашТокенВідTMDB
```

Як отримати токен від TMDB:
- Перейдіть на [https://www.themoviedb.org/](https://www.themoviedb.org/)
- Зареєструйтеся або увійдіть до облікового запису
- Перейдіть до Settings → API
- Скопіюйте ваш API Token (Bearer Token)

**Важливо для Vercel:** 
Якщо ви розгортаєте на Vercel, слід також встановити змінну оточення через дашборд Vercel:
1. Перейдіть до проєкту на [vercel.com](https://vercel.com)
2. Settings → Environment Variables
3. Додайте нову змінну:
   - Name: `VITE_TMDB_TOKEN`
   - Value: `вашТокен`
4. Перерозгорніть проєкт після додавання змінної

4. **Запуск dev-сервера**
```bash
npm run dev
```
Додаток буде доступний за адресою `http://localhost:5173/`

5. **Збірка для продакшену**
```bash
npm run build
```

6. **Перегляд продакшену збірки**
```bash
npm run preview
```

## 📖 Компоненти

### SearchBar
- Форма пошуку з полем введення та кнопкою
- Валідація: перевіряє, що користувач ввів текст
- Сповіщення про помилку при спробі пошуку з пустим полем

### MovieGrid
- Галерея карток фільмів
- Ліниве завантаження зображень (`loading="lazy"`)
- Реагує на клік користувача для відкриття модального вікна

### Loader
- Індикатор завантаження під час запиту до API
- Показується під час обробки HTTP-запиту

### ErrorMessage
- Повідомлення про помилку
- Показується, якщо запит до API не вдався

### MovieModal
- Модальне вікно з детальною інформацією про фільм
- Закриється при:
  - Натисканні на кнопку ✕
  - Натисканні клавіші ESC
  - Кліку за межами модального вікна
- Використовує React Portal для рендеру поза основним DOM
- Блокує прокручування сторінки під час відкриття

## 🔌 API Інтеграція

Проєкт використовує [The Movie Database (TMDB) API](https://www.themoviedb.org/settings/api) для отримання даних про фільми.

**Запит:**
```
GET https://api.themoviedb.org/3/search/movie
```

**Параметри:**
- `query` - ключове слово для пошуку
- `include_adult` - включати фільми для дорослих
- `language` - мова результатів
- `page` - номер сторінки

**Авторизація:**
Токен передається через заголовок `Authorization: Bearer {token}`
